

## Instalation

- create and edit .env file
- composer update
- php artisan key:generate
- php artisan passport:install
- php artisan migrate
- php artisan db:seed
- php artisan storage:link
- php artisan l5-swagger:generate
- chmod 777 -R storage
- npm install
- npm run dev
